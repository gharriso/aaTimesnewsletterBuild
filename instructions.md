Each week, I have to create a newsletter from the contents of aatimes.org.au.  The file `sampleNewsletter.docx` is an example.  

There are some early attempts to automate the process here.  What you need to do is:

1.  Update the "New Meetings", "Meetings changed" and "Meetings no longer functioning" with the entries on aatimes.org.au for changes that are highlighted
2.  Update the rest of the first page with events starting the upcoming Monday
3.  Place images on the second page corresponding to those events or other events in the future.  
4.  Update the date in the newsletter header. 
5.  If an image for an event on the first page occurs ont he second page, place a "See Next Page" under the date on the first page.

Try and keep the formatting and layout as close to the example as possible.  If you can't fit all the events for the next week on the first page, then overflow and I will fix up.
